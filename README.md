# 🌸 Bloomumu

**A Premium, Interactive Romantic Wishing SaaS**

Bloomumu is a minimalist, glassmorphism-styled platform designed to create unforgettable digital surprises for your loved ones. Built as a digital love letter from **Ehsanul Islam Sajjad** to **Tasnim Bintay Anwar Mumu**, Bloomumu combines high-end tech with deep emotional storytelling.

### ✨ Features
- **Dynamic Personalization**: Automatically generates unique magic links for your partner via URL parameters.
- **SaaS Lead Funnel**: Built-in email capture that logs leads directly to a Google Sheet via serverless webhooks.
- **Premium Aesthetics**: High-end glassmorphism design with interactive heart-particle systems.
- **Tactile Interaction**: Features a custom "Seal the Promise" hold-to-unlock mechanism with synthesized Web Audio feedback.
- **Social Ready**: Optimized with OpenGraph tags and romantic sharing presets for WhatsApp, Facebook, and Instagram.

### 🛠️ Tech Stack
- **Frontend**: Vanilla HTML5 / CSS3 (Custom Glassmorphism & Keyframe Animations)
- **Logic**: JavaScript (Web Audio API & URLSearchParams handling)
- **Backend**: Google Apps Script (Serverless lead capture to private spreadsheets)

### 🚀 Getting Started
1. Clone the repository.
2. Upload all files (including `og_image_romantic_wish.png`) to any static host (GitHub Pages, Netlify, or Vercel).
3. Connect your own Google Apps Script URL in `index.html` to start collecting leads.
